# notes-site-cn

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。它不针对任何特定域名或网站，而是作为一个通用的页面存储与展示空间，便于将不同主题的静态页面集中管理、版本追踪与在线访问。

## 目录说明

```
notes-site-cn/
├── index.html          # 仓库首页/入口页面
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── example1.html   # 示例页面 1
│   ├── example2.html   # 示例页面 2
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`: 所有独立 HTML 页面存放于此，每个页面为一个独立的归档单元。
- `assets/`: 页面引用的公共资源，按类型分目录存放。

## 页面归档说明

- 每个 HTML 页面均为独立归档，可包含完整的样式与脚本，不依赖外部服务。
- 页面内容以信息记录、学习笔记、技术演示等非商业用途为主。
- 归档内容不涉及具体域名、推广信息或诱导点击行为。
- 页面文件名建议使用语义化的英文或拼音，避免特殊字符。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 以新增、修正或优化页面。
- 添加新页面时，请保持目录结构清晰，并在 `pages/` 下放置文件。
- 如需更新公共资源，请确保不影响已有页面的正常显示。
- 本仓库不保证长期稳定在线，请自行备份重要内容。
